# Elijah Olusegun — Portfolio

My personal portfolio website built with React, React Router, Tailwind CSS and Zustand.

## Pages
- Home — intro, skills and CV download
- About — bio, education, certificates and technical skills
- Projects — all deployed projects with live and GitHub links
- Contact — email, GitHub, LinkedIn and contact form

## Features
- Dark and light mode toggle
- Fully responsive on all screen sizes
- React Router for multi-page navigation
- Zustand for global theme state
- All projects linked to live deployments

## Built with
- React
- React Router
- Tailwind CSS
- Zustand
- Vite
